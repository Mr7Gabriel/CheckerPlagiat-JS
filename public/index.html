🎯 Aplikasi Pengecek Plagiat Multi-Format

Sistem deteksi plagiarisme canggih dengan dukungan berbagai format dokumen dan akurasi tinggi menggunakan algoritma NLP terdepan.

🧠
Algoritma Canggih
Menggunakan kombinasi 3 algoritma: Cosine Similarity, N-gram Analysis, dan Fingerprinting untuk deteksi yang akurat
📁
Multi-Format Support
Mendukung berbagai format: TXT, PDF, DOC, DOCX, ODT, RTF dengan ekstraksi teks otomatis
⚡
Real-time Processing
Analisis cepat dengan hasil instan, interface responsif, dan progress tracking
📊
Detailed Reports
Laporan lengkap dengan breakdown per algoritma, contoh frasa yang cocok, dan visualisasi hasil

📊 Algoritma yang Digunakan:

Cosine Similarity - Kesamaan vektor teks 40%
N-gram Analysis - Urutan kata dan frasa 35%
Fingerprinting - Pola unik teks 25%

🎯 Tingkat Akurasi:

🟢 AMAN (0-24%) - Tidak ada indikasi plagiat
🟡 RENDAH (25-49%) - Perlu review lebih lanjut
🟠 SEDANG (50-79%) - Perlu revisi signifikan
🔴 TINGGI (80-100%) - Plagiat terdeteksi, perlu penulisan ulang

⚙️ Teknologi:

  • Backend: Node.js + Express.js
  • Frontend: HTML5, CSS3, JavaScript
  • File Processing: Multer + Mammoth + PDF-Parse
  • Text Analysis: Custom NLP algorithms
  • UI/UX: Responsive design dengan real-time feedback
